# MIUI 12 Font Module

This Magisk module provides MIUI 12 fonts for you Android AOSP interface

![](https://i.imgur.com/cKSrKFB.png)
 
## Requirements 
- Android 8.0+ 
- Magisk 19+ 

## Installation 
1. Flash this module in Magisk Manager. 
2. Reboot device. 
3. Enjoy!

- LOS ROM
